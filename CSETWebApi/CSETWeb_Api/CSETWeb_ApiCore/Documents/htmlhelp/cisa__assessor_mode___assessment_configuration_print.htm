CISA  Assessor Mode - Assessment Configuration

 
Selecting Assessment Configuration within the Prepare menu displays the Assessment Configuration page. This page is only available if CISA Assessor Workflow mode is enabled. All required fields must be completed to access the assessment's reports and exports.
 

Assessment and Organization Details

  • The assessment and organization details section includes the following fields:
  • Assessment Name
  • A read-only field displaying a system-generated assessment name.
  • If the PCII Requested field is selected as "No," the name displays in the following format:
  • [Assessment Type].non-pcii.[Date of Assessment]-[Extension]
  • If the PCII Requested field is selected as "Yes," the name displays in the following format:
  • [Assessment Type].pcii.[PCII Number]
  • The assessment name will display on the My Assessments tab, the assessment's reports, and will be used as the default assessment file name upon export.
  • Assessment Date
  • Enter the date of the assessment.
  • Displays the current date by default.
  • Selecting the field displays the date picker.
  • City or Site Name
  • Enter the city or site name where the assessment is to be performed.
  • State/Province/Region
  • Enter the state, province, or region where the assessment is to be performed.
  • CISA Region
  • Select the CISA region from the dropdown list where the assessment is to be performed.
  • PCII Requested
  • Select the Yes button to display the PCII Number field.
  • Upon selection, a green banner displays at the top of CSET and includes a link to learn more about Protected Critical Infrastructure Information.
  • The No button is selected by default.

Assessment Type

  • The Assessment Type section displays a brief description of the standard or model.

Contacts

  • The Contacts section displays a list of personnel associated with the assessment.
  • Users can add contacts by selecting the Add Contacts button.
  • Once added, the contact card will display.
  • Select the Change button to edit the contact card.
  • Select the Email button to open the Invitation Email window to invite the user to participate in the assessment.
  • Select the Remove button to delete the contact card.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the contact.
  • Select the No button to keep the contact and close the dialog box.
  • The user who created the assessment is automatically listed as the Assessment Owner.
  • This contact information cannot be edited.

Add Contacts

  • Select the Add Contacts button to associate additional personnel with the assessment.
  • Upon selection, the Add Contacts form displays.
  • Enter the applicable information for each field.
  • Select one of the following roles that best applies to the contact:
  • Participant
  • Participates in the assessment but cannot add additional contacts.
  • Facilitator
  • Facilitates the assessment and can add additional contacts.
  • Select the Save button to add the contact.
  • Select the Cancel button to disregard any changes and close the Add Contacts form.

Critical Service Details

  • The critical service details section includes the following fields:
  • Facilitator
  • The Facilitator dropdown list includes all contacts associated with the assessment.
  • Select the facilitator who will be performing the assessment from the dropdown list.
  • Select the checkbox if the facilitator is conducting a self assessment.
  • If selected, a "SELF ASSESSMENT" label will display on the report cover.
  • Critical Service Name
  • Enter the name of the critical service.
  • Critical Dependency for Incident Response Support
  • Enter the critical dependency required to support incident response.
  • Critical Service Point of Contact
  • Select the critical service point of contact from the dropdown menu.
  • The dropdown menu includes a list of all contacts added to the assessment.
  • Please estimate the percentage of your organization's annual revenue that depends on the critical service.
  • Select the percentage range from the dropdown menu that best aligns with the organization's annual revenue that depends on the critical service.
  • If possible, please estimate the number of people served annually by the critical service.
  • Select the numerical range from the dropdown menu that best aligns with the number of people served annually by the critical service.
  • Please select in order of criticality, the two critical infrastructure sectors which, if they were unavailable or disrupted for one week or more, would most affect your ability to sustain the critical service.
  • Select the most important and second most important critical infrastructure sectors from the dropdown menus.
